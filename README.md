# Sweet-Dairy-Family-Farm

A website for our family-run farm in McDonough, Georgia, sharing our story, our animals, and the free-range eggs and fresh milk we produce.

Taking inspiration from local farms.

Currently doing this in tandem with a Web Dev course at WGU to enforce foundations.

## Site structure

| File | Purpose |
| --- | --- |
| `index.html` | Home page — hero, highlights, what we offer, why choose us |
| `about.html` | Our story, Meet Kimberly, our values |
| `products.html` | Eggs & milk, the honor-system farm shop, wholesale |
| `contact.html` | Contact cards, message form, map |
| `styles.css` | All styling — design tokens, components, responsive rules |
| `script.js` | Mobile nav toggle, scroll-reveal animations, contact form |
| `images/` | Stock photos (see credits below) |

Built with plain HTML, CSS, and vanilla JavaScript — no frameworks or build tools.
